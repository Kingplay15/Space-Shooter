# Unity Space Shooter

A 2D space shooter developed in Unity as part of my learning journey.

This project started from a Unity course and was later expanded with additional gameplay systems to transform it into a complete mini game with progression and RPG-inspired mechanics.

## Features

### Gameplay

* Player spaceship movement
* Enemy waves
* Shooting system
* Health and game over system
* Score tracking

### RPG Elements

* Multiple bullet types
* Shield system
* Player health upgrades
* Progressive gameplay loop

### User Interface

* Main menu
* In-game HUD
* Game over screen

## Technical Highlights

* Object spawning system
* Projectile management
* Health and damage system
* Simple gameplay progression
* Modular player weapon system

## Controls

* W/A/S/D : Move
* Left Mouse : Shoot

## Learning Objectives

This project helped me practice:

* Unity gameplay programming
* Prefabs and object spawning
* UI implementation
* Player progression systems
* Expanding an existing project with new gameplay features
